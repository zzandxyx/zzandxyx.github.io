Linux编程入门

使用技巧

  • vim显示行号
  • 配置vscode连接远程服务器不需要每次输入密码
    • win终端输入,生成公钥：id_rsa.pub
    • 服务器端在home目录下，进入.ssh文件夹
    • 把win下id_rsa.pub中的内容复制到上面文件里，保存退出

1、静态库制作

  • 命名规则：
    • Linux:libxxx.a
    • lib:前缀（固定）
    • xxx：库名
    • .a:后缀（固定）
  • 制作：
    • gcc获得.o文件
    • 将.o文件打包，使用ar工具（archive）
    • r：将文件插入备存文件中
    • c：建立备存文件
    • s：索引

2、动态库制作

  • 命名规则：
    • Linux：libxxx.so
    • lib：前缀
    • xxx：库名
    • .so：后缀
    • 在linux下是一个可执行文件
  • 制作：
    • gcc得到.o文件，得到和位置无关的代码
    • gcc得到动态库
  • 查看动态库依赖关系

3、makefile

  • 文件命名
    • makefile or Makefile
  • 规则
    • 一个Makefile文件可以有一个或多个规则
    • 目标 …:依赖…
      命令（shell命令）
      …
      • 目标：最终要生成的文件
      • 依赖：生成目标所需要的文件或是目标
      • 命令：执行命令对依赖操作生成目标（命令签必须Tab缩进）
    • makefile中的其他规则都是为第一条规则服务的
  • 工作原理
    • 检测更新，执行规则命令时，会比较目标和依赖文件的时间
      • 如果依赖的时间比目标时间晚，需要重新生成目标
      • 相反，目标不需要更新，对应规则中的命令不需要被执行

4、GBD调试

  • 生成可执行程序
  • 启动
  • 退出
  • 给程序设置参数
  • 获取设置参数
  • 查看当前源代码 list/l xxx.cpp:行号
    list/l xxx.cpp:函数名 show listsize
    set listsize break/b 行号
    break/b 函数名
    break/b 文件名：行号
    break/b 文件名：函数 info/i break/b d/del/delete 断点编号 enable/disable 断点编号 b 10 if i==5 run start c/continue n/next p/print 变量名（打印变量值）
    ptype 变量名（打印变量类型） until finish
